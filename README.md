## 📌 Theme Explanation

**EduLearn** is an online learning platform designed to make quality education accessible to everyone. The platform offers courses in Technology, Design, Business, and Science — with a focus on practical skills, verified certificates, and real community support.

Target Audience: Students, working professionals, and beginners looking to upskill.

## ✨ Features List

### Pages
- ✅ **Homepage** – Hero section, animated course cards, features, stats, testimonial
- ✅ **About Page** – Mission, team members, company timeline
- ✅ **Showcase Page** – 6 courses with live category filter (Tech / Design / Business / Science)
- ✅ **Interactive Page** – Login/Signup forms, 5-question Quiz, Progress Tracker
- ✅ **Contact Page** – Validated contact form, social links, Google Map
- ✅ **Blog Page** – 4 articles with search, expandable read more

### Technical Features
- 🌙 **Dark / Light Mode Toggle** (saved in localStorage)
- 📋 **Form Validation** – Login, Signup, Contact (with live error messages)
- 🎯 **Course Filter** – Filter by category with animation
- 🧠 **Interactive Quiz** – 5 questions, score display, retry option
- 📈 **Progress Tracker** – Update course progress with a slider
- 🔍 **Blog Search** – Real-time article filtering
- 📱 **Responsive Design** – Mobile, tablet, and desktop ready
- 🍔 **Hamburger Menu** – Mobile navigation
- 🎞️ **Scroll Animations** – Fade-in on scroll using IntersectionObserver
- 📖 **Read More Toggle** – Expandable blog articles

## ⚙️ How to Run Locally

```bash
# Option 1 – Just open in browser
Double-click index.html

# Option 2 – Python server
cd EduLearn_Project
python -m http.server 8000
# Visit: http://localhost:8000

# Option 3 – VS Code Live Server
Right-click index.html → Open with Live Server
```

## 📤 How to Deploy on GitHub Pages

```
1. Create a GitHub account at https://github.com
2. Click "New Repository" → Name it: EduLearn_Project
3. Upload all files (maintain folder structure)
4. Go to Settings → Pages
5. Source: Deploy from branch → main → / (root)
6. Click Save → Your live link appears in ~2 minutes
```
